# File Organization System

Uma sistema de organização de arquivos completo que analisa computadores, sugere melhorias na organização de arquivos e cria layouts profissionais com integração na nuvem.

## 🚀 Características Principais

- **Análise Completa do Sistema**: Análise abrangente de hardware, software e estrutura de arquivos
- **Organização Inteligente**: Categorização automática e sugestões de organização profissional
- **Layouts Profissionais**: Criação de layouts limpos, dinâmicos e profissionais
- **Integração na Nuvem**: Suporte para AWS S3, Google Cloud Storage e Azure Blob Storage
- **Gestão de Memória Virtual**: Arquivamento e sincronização inteligente na nuvem

## 📋 Requisitos

- Python 3.7+
- Dependências listadas em `requirements.txt`

## 🛠️ Instalação

1. Clone o repositório:
```bash
git clone https://github.com/operatortecnick/X-v-deossss.git
cd X-v-deossss
```

2. Instale as dependências:
```bash
pip install -r requirements.txt
```

## 🎯 Uso

### Comandos Principais

**Analisar Sistema:**
```bash
python main.py analyze --depth comprehensive --report
```

**Organizar Arquivos (Preview):**
```bash
python main.py organize --path ./minha-pasta
```

**Organizar Arquivos (Aplicar Mudanças):**
```bash
python main.py organize --path ./minha-pasta --apply --backup
```

**Criar Layout Profissional:**
```bash
python main.py layout --style professional --theme modern --apply
```

**Integração na Nuvem:**
```bash
python main.py cloud --provider aws --upload --sync
```

**Workflow Completo:**
```bash
python main.py complete --path ./minha-pasta --apply --backup
```

### Verificar Status do Sistema

```bash
python main.py status
```

### Ajuda

```bash
python main.py --help
python main.py [comando] --help
```

## ⚙️ Configuração

Edite o arquivo `config.yaml` para personalizar:

- Profundidade de análise
- Categorias de arquivos
- Regras de organização
- Configurações de nuvem

## 📊 Funcionalidades

### Análise do Sistema
- Análise de hardware (CPU, memória, armazenamento)
- Análise de software (SO, processos, serviços)
- Análise de arquivos (estrutura, tipos, duplicatas)
- Análise de performance (uso de recursos)

### Organização de Arquivos
- Categorização automática por tipo
- Organização por data
- Detecção e remoção de duplicatas
- Otimização de nomes de arquivos
- Backup automático antes das mudanças

### Layouts Profissionais
- **Estilos**: Professional, Minimal, Dynamic
- **Temas**: Modern, Classic, Corporate
- Ícones customizados
- Atalhos automáticos
- Organização da área de trabalho

### Integração na Nuvem
- Suporte multi-nuvem (AWS, Google Cloud, Azure)
- Compressão automática
- Sincronização inteligente
- Gestão de memória virtual
